
## 6. Functions

A function is a named block of code for reuse and better structure.

```bash
#!/bin/bash

# Declaration
log_message() {
    local timestamp=$(date '+%Y-%m-%d %H:%M:%S')
    echo "[$timestamp] $*"
}

# Calling with arguments
log_message "Starting server setup"
```

- `local` limits variable scope to the function.
- Inside a function, arguments are `$1`, `$2`, etc. (their own, don't confuse with script arguments).

Returning a value:
```bash
check_root() {
    if [[ $EUID -eq 0 ]]; then
        return 0  # success
    else
        return 1  # failure
    fi
}

if check_root; then
    echo "Running as root"
else  
    echo "Running as non-root"
fi
```

`return` only gives a numeric exit code (0 = true/success). To return a string, use `echo` and capture: `result=$(some_func)`.
