## 5. Loops

### 5.1. `for` Loop

#### Iterate over a list of words

```bash
for user in alice bob charlie; do
    echo "Adding user $user"
    echo "simulate command useradd -m ${user}"
done
```

#### C-style loop

```bash
for ((i=0; i<=5; i++)); do
    echo "Step $i"
done
```

#### Over files

```bash
for script in /etc/init.d/*; do
    echo "Service script: $(basename $script)"
done
```

### 5.2. `while` Loop
```bash
count=0
while [[ $count -le 3 ]]; do
    echo "Iteration $count"
    ((count++))
done
```

A typical pattern: reading a file line by line

```bash
while IFS= read -r line; do
    echo "Line: $line"
done < ./5.Loops.md
```

`IFS=` prevents trimming of whitespace, `-r` disables backslash interpretation.

### 5.3. Infinite Loop and `until`

`until` runs as long as the condition is false:

```bash
until ping -c1 8.8.8.8 &>/dev/null; do
    echo "Waiting for network..."
    sleep 5
done
echo "Network is up!"
```

### 5.4. Loop Control: `break`, `continue`
```bash
for file in *.md; do
    [[ -f "$file" ]] || continue  # skip if not a file
    if [[ $file == "5.Loops.md" ]]; then
        echo "Processing interrupted at $file"
        break
    fi
    echo "Archiving $file"
done
```
