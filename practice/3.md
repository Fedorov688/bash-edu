## Task 3: Input, Output and Redirection

**Description**  
Create a script that:
- Asks the user for a filename (with prompt `"Enter filename: "`).
- Reads the filename.
- Checks if the file exists. If it does, append the current date and time to it in the format `YYYY-MM-DD HH:MM:SS` (see `man date`).
- If the file does not exist, write the date and time to a new file with that name, but send any error messages to `/dev/null`.
- Write a short report to the terminal.

**Input Data**  
No external file is needed; the script creates or appends. Assume the user enters `mylog.txt`.

**Expected Result**  
If `mylog.txt` exists, the script appends a timestamp line. If not, it creates the file. And script should print something like:
```
---- Report ----
Date written to mylog.txt
----------------
```
