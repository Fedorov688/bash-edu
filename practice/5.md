## Task 5: Loops

**Description**  
Create a script that iterates over all `.conf` files in `/etc/` (without recursing) and does the following:
- Skips any file that is not readable (just continue).
- For each file, count the number of lines that contain the word `error` (case‑insensitive).
- Print `"<filename>: <count> error(s)"`.
- Stop processing if a file named `sysctl.conf` is encountered.

**Input Data**  
The script runs on a standard Linux system. No extra input.

**Expected Result**  
An output like:
```
resolv.conf: 0 error(s)
ca-certificates.conf: 0 error(s)
...
```
If `sysctl.conf` exists, the loop stops when it’s reached. Otherwise, all `.conf` files are processed.
