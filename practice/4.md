## Task 4: Conditional Statements

**Description**  
Write a script that takes one argument: a file path.
- If the argument is not provided, print `"Usage: $0 <file>"` and exit with code 1.
- If the path is a directory, print `"<path> is a directory."`
- If the path is a regular file that is readable and non‑empty, print `"<path> is a readable, non‑empty file."`
- Otherwise, print `"<path> does not exist or is not accessible."`

Use the extended test `[[ ... ]]`.

**Input Data**  
Run the script with various arguments, e.g.:
```
./check.sh /etc/passwd
./check.sh /tmp
./check.sh /nonexistent
```

**Expected Result**  
For `/etc/passwd` you should see: `/etc/passwd is a readable, non‑empty file.`  
For `/tmp` (a directory): `/tmp is a directory.`  
For missing file: `/nonexistent does not exist or is not accessible.`
