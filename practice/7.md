## Task 7: Text Processing Pipeline

**Description**  
Given a log file (you can create a sample file) that contains lines like:
```
192.168.1.10 - - [10/May/2026:14:32:10 +0000] "GET /index.html HTTP/1.1" 200 4523
10.0.0.5 - - [10/May/2026:14:32:11 +0000] "POST /login HTTP/1.1" 401 128
192.168.1.10 - - [10/May/2026:14:32:12 +0000] "GET /about.html HTTP/1.1" 200 2300
...
```
Write a script that:
1. Uses `grep` to extract only lines with HTTP status 200.
2. Uses `awk` to compute the total bytes sent (the last column) for those 200‑status requests.
3. Uses `sed` in the pipeline to remove any trailing whitespace before passing to `awk`.
4. Prints `"Total bytes for 200 requests: <value>"`.

**Input Data**  
Create a test file `access.log` containing:
```
192.168.1.10 - - [10/May/2026:14:32:10 +0000] "GET /index.html HTTP/1.1" 200 4523
10.0.0.5 - - [10/May/2026:14:32:11 +0000] "POST /login HTTP/1.1" 401 128
192.168.1.10 - - [10/May/2026:14:32:12 +0000] "GET /about.html HTTP/1.1" 200 2300
10.0.0.7 - - [10/May/2026:14:32:13 +0000] "GET /contact HTTP/1.1" 200 5120
```

**Expected Result**
```
Total bytes for 200 requests: 11943
```
(4523 + 2300 + 5120 = 11943)
