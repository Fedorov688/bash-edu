## Task 2: Variables and Arguments

**Description**  
Write a script that accepts three command-line arguments: a username, a shell (e.g., `/bin/bash`), and a home directory. The script must:
- Print a message: `"Creating user <username> with shell <shell> and home <home>"`
- Create a variable `backup_dir` containing the home path with `_backup` appended.
- Print the backup directory name.

**Input Data**  
Run the script like this:
```
./script.sh john /bin/zsh /home/john
```

**Expected Result**
```
Creating user john with shell /bin/zsh and home /home/john
Backup directory: /home/john_backup
```
