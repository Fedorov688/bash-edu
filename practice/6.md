## Task 6: Functions (Section 6)

**Description**  
Write a script that defines two functions:
- `is_number()` – accepts one argument and returns 0 if it consists only of digits, 1 otherwise.
- `square()` – accepts one argument, calculates its square (`$1 * $1`) and **echoes** the result.
  In the main part of the script, read a value from the user, call `is_number` to validate it, and if valid, call `square` to compute and print the square. If invalid, print an error.

**Input Data**  
User enters `7`.

**Expected Result**
```
Enter a number: 7
The square of 7 is 49.
```
If the user enters `abc`, output: `"Error: not a number."`
