## Task 1: Your First Script

**Description**  
Create a Bash script that prints a greeting and the current system's hostname and kernel version.

**Input Data**  
None – the script should gather information using commands.

**Expected Result**  
When you execute the script, the output should look similar to:
```
Hello, Admin!
Hostname: myserver
Kernel: 5.15.0-91-generic
```
(The actual hostname and kernel version will depend on your machine.)
